
## About Posty

Posty is a lightweight social posting platform built with Laravel 12. It allows users to register, log in, create short posts, like or unlike posts, and browse user profiles.

This project is focused on clean Laravel architecture and maintainable UI patterns, including reusable Blade components for forms, buttons, alerts, and cards.

### Core Features

- Authentication (register, login, logout)
- Public post feed with pagination
- Create and delete posts (with authorization)
- Like and unlike posts
- User profile pages with post and like stats
- Responsive, component-based UI using Tailwind CSS + Vite

### Tech Stack

- Laravel 12
- PHP 8.2
- MySQL
- Blade templates + reusable view components
- Tailwind CSS v4
- Vite
